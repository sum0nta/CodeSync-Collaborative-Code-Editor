═══════════════════════════════════════════════════════════════

STEP 1: SETUP DATABASE (5 minutes)
═══════════════════════════════════════════════════════════════
□ 1. Go to: https://www.mongodb.com/cloud/atlas/register
□ 2. Sign up with Google account
□ 3. Create FREE M0 Cluster (512MB)
     - Provider: AWS
     - Region: Choose closest to you
     - Name: codesync-cluster
□ 4. Database Access → Add Database User
     - Username: codesync-admin
     - Password: (generate & SAVE!)
     - Role: Atlas admin
□ 5. Network Access → Add IP Address
     - Click "Allow Access from Anywhere" (0.0.0.0/0)
□ 6. Database → Connect → Drivers → Copy connection string
     - Replace <password> with your password
     - Add /codesync before the ?
     
     Example:
     mongodb+srv://codesync-admin:YOUR_PASSWORD@codesync.xxxxx.mongodb.net/codesync?retryWrites=true&w=majority

✅ SAVE THIS CONNECTION STRING! You'll need it in Step 2.

═══════════════════════════════════════════════════════════════

STEP 2: DEPLOY BACKEND (5 minutes)
═══════════════════════════════════════════════════════════════
□ 1. Go to: https://render.com/
□ 2. Sign up with GitHub account
□ 3. Authorize Render to access your repositories
□ 4. Click "New +" → "Web Service"
□ 5. Connect your CodeSync repository
□ 6. Configure:
     - Name: codesync-backend
     - Region: Oregon (US West)
     - Branch: main
     - Root Directory: backend
     - Environment: Node
     - Build Command: npm install
     - Start Command: npm start
     - Plan: Free
□ 7. Click "Environment" and add these variables:
     
     NODE_ENV=production
     PORT=10000
     MONGODB_URI=(paste your MongoDB connection string from Step 1)
     JWT_SECRET=(generate at https://www.random.org/passwords/?num=1&len=64&format=plain)
     
     Optional (for AI features):
     GOOGLE_AI_API_KEY=(get from https://aistudio.google.com/app/apikey)
     
□ 8. Click "Create Web Service"
□ 9. Wait 5-10 minutes for deployment to complete

✅ Your backend URL will be: https://codesync-backend.onrender.com
✅ Test it: Visit https://YOUR-BACKEND.onrender.com/api/health
   Should return: {"status":"OK",...}

═══════════════════════════════════════════════════════════════

STEP 3: DEPLOY FRONTEND (5 minutes)
═══════════════════════════════════════════════════════════════
□ 1. Go to: https://vercel.com/signup
□ 2. Sign up with GitHub account
□ 3. Authorize Vercel
□ 4. Click "Add New Project"
□ 5. Import your CodeSync repository
□ 6. Configure:
     - Framework Preset: Create React App
     - Root Directory: frontend
     - Build Command: npm run build
     - Output Directory: build
□ 7. Click "Environment Variables" and add:
     
     REACT_APP_BACKEND_URL=(Your backend URL from Step 2)
     
     Example: https://codesync-backend.onrender.com
     
□ 8. Click "Deploy"
□ 9. Wait 2-3 minutes for deployment

✅ Your frontend URL will be: https://codesync-XXXXX.vercel.app

═══════════════════════════════════════════════════════════════

STEP 4: CONNECT FRONTEND & BACKEND (2 minutes)
═══════════════════════════════════════════════════════════════
□ 1. Go back to Render.com dashboard
□ 2. Click on your backend service
□ 3. Go to "Environment" tab
□ 4. Add new environment variable:
     
     FRONTEND_URL=(Your Vercel URL from Step 3)
     
     Example: https://codesync-xxxxx.vercel.app
     
□ 5. Click "Save Changes"
     (Render will automatically redeploy - takes 2-3 minutes)

🐛 TROUBLESHOOTING:
═══════════════════════════════════════════════════════════════
Issue: Backend won't start
Fix: Check MongoDB connection string in Render env vars

Issue: Frontend shows connection errors
Fix: Verify REACT_APP_BACKEND_URL is correct in Vercel

Issue: 502/503 errors
Fix: Backend is waking up from sleep, wait 30-60 seconds

Issue: CORS errors
Fix: Make sure FRONTEND_URL is set in Render backend env
